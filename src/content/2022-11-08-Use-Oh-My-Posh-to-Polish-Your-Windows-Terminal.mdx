目前你用简体中文搜索到 Oh My Posh 安装方法大概都用了这一条命令：`Import-Module oh-my-posh`

事实上这种方式已经不再被 Oh My Posh 官方支持了，当然你可以用这个方法安装，只不过每次打开你的 Windows Terminal 都会弹出一大串提示信息，官方已经在网站上说明了

https://ohmyposh.dev/docs/migrating

如果英语还算行的话，直接看着官网的文档安装就行了。但是既然你已经搜索到这里，那么...

## Installation
Windows 直接用 winget 安装就行了。运行下面这条命令，过一会就装上了。如果长时间没反应，那就是网络问题了（墙国特色)

```shell
winget install JanDeDobbeleer.OhMyPosh -s winget
```

## Configuration
运行下面这条命令创建 pwsh 的 profile 文件

```shell
New-Item -Path $PROFILE -Type File -Force
```

最好还是用VSCode去编辑该文件，如果你下载了VSCode，但是 code 命令没有反应的话，搜索一下“添加VSCode到环境变量”接口。

然后用 `code $PROFILE` 这条命令，在文件内写入 `oh-my-posh init pwsh --config ~/jandedobbeleer.omp.json | Invoke-Expression` 保存即可。如果无法使用 code 请确保已经将 vscode 添加到环境变量。

如果想要其他的主题，在官网下载好相关的 json 之后，将 ` ~/.jandedobbeleer.omp.json`替换为你下载下来的 json 文件的路径就行了。

## 配置 PowerShell 字体
个人使用的是FiraCode Nerd Font这个字体。下载之后，在 `Windows Terminal - 设置 - PowerShell - 外观` 里面改变字体就行。
